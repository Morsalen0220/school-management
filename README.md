## How to install

- Unzip the folder inside your site root.
- Load home page `http://localhost`
- Enter credentials and submit
- When you see installing done, reload page - goto `http://localhost/create-super-admin` to create your super admin account

NOTE: localhost is a local address, you should replace it with your own domain or IP address.
